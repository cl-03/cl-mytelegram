**版本**: 0.30.0

## 一、开发目标

### 1.1 Bot API 9.7+ 支持

跟进 Telegram Bot API 最新更新（2026 Q2）：

- **位置 API** - 设备地理位置访问
- **联系人 API** - 设备联系人集成
- **通知 API** - 浏览器推送通知
- **文件选择器** - 设备文件选择上传

### 1.2 Mini App 功能增强

扩展 v0.29.0 的 Mini App 功能：

- **Haptic Feedback** - 触觉反馈 API
- **Biometric Auth** - 生物识别认证
- **Clipboard API** - 剪贴板访问
- **Share Target** - 分享目标处理

### 1.3 性能优化

- **连接池** - MTProto 连接池管理
- **请求批处理** - 批量 RPC 调用
- **增量更新** - 增量数据同步

## 三、实现计划

### 阶段 1: Bot API 9.7 核心功能（Day 1 上午）

**文件**: `src/api/bot-api-9-7.lisp`

#### 3.1.1 位置 API

```lisp
;; 请求位置访问
(defun request-location-access ()
  "Request device location access via CLOG")

;; 获取当前位置
(defun get-current-location (&key high-accuracy timeout maximum-age)
  "Get current device location")

;; 监听位置变化
(defun watch-position (callback &key enable-high-accuracy)
  "Start watching position changes")

;; 停止监听
(defun clear-position-watch (watch-id)
  "Stop watching position")
```

#### 3.1.2 文件选择器

```lisp
;; 选择文件
(defun select-files (&key accept multiple max-file-size)
  "Open file picker and return selected files")

;; 选择目录
(defun select-directory (&key recursive)
  "Open directory picker")

;; 读取文件内容
(defun read-file-content (file-handle &key as-type)
  "Read file content as text, base64, or array buffer")
```

#### 3.1.3 通知 API

```lisp
;; 请求通知权限
(defun request-notification-permission ()
  "Request browser notification permission")

;; 发送通知
(defun send-notification (title &key body icon badge tag require-interaction)
  "Show browser notification")

;; 监听通知点击
(defun on-notification-click (handler-id handler-fn)
  "Register notification click handler")
```
