# v0.26.0 开发计划 - 群组视频通话增强

**版本**: 0.26.0  
**发布日期**: 2026-05-04  
**优先级**: P1 - 重要用户体验功能

---

## 一、功能概述

### 1.1 群组视频通话增强

在现有 voip.lisp 和 webrtc-ffi.lisp 基础上，实现完整的多人视频通话功能。

**核心功能**:
- 支持 10+ 人同时视频通话
- 屏幕共享功能
- 通话质量自适应（根据带宽调整分辨率/帧率）
- 通话录制功能
- AI 降噪处理

### 1.2 视频消息

实现 Telegram 风格的圆形视频消息。

**核心功能**:
- 圆形视频消息格式（MP4/H.264）
- 视频压缩和编码
- 录制进度显示
- 发送/接收/播放

### 1.3 媒体合集管理

改进媒体文件管理体验。

**核心功能**:
- 智能相册创建（按时间/事件/标签）
- 媒体标签系统
- 快速搜索和过滤
- 媒体文件导出
