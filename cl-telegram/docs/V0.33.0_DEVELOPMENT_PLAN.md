### 2.1 Stories API 完成 (优先级：高)

**文件**: `src/api/stories-complete.lisp` (~400 行)  
**测试**: `tests/stories-complete-tests.lisp` (~100 行)

| 功能 | 描述 | 函数数 |
|------|------|--------|
| sendStory | 发送故事到频道/个人 | 8 |
| getStory | 获取故事详情 | 4 |
| editStory | 编辑已发送故事 | 6 |
| deleteStory | 删除故事 | 4 |
| getStories | 批量获取故事 | 4 |
| storyViews | 查看统计 | 4 |

**新增函数**: 30+

#### API 示例
```lisp
;; 发送故事
(send-story chat-id media :caption "My story" :privacy :contacts)

;; 编辑故事
(edit-story story-id :caption "Updated caption" :add-media additional-photo)

;; 获取故事统计
(get-story-views story-id)
```

**新增函数**: 42+

#### API 示例
