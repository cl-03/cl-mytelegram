## 三、v0.31.0 开发优先级

### 优先级 1: 完善核心缺失功能 (必做)

#### 1.1 文件管理完善 (高优先级)

**现状**: 已有基础文件选择器，但缺少：
- 大文件分片上传
- 文件下载管理
- 媒体文件元数据提取
- 文件缓存管理

**新增函数**:
```lisp
;; 文件上传
(upload-file file-path &key chunk-size callback)
(upload-file-chunk chunk-data chunk-index total-chunks)
(get-upload-status upload-id)

;; 文件下载
(download-file file-id &key destination callback)
(download-file-part file-id offset limit)
(get-download-progress download-id)

;; 媒体管理
(get-media-metadata file-id)
(extract-video-thumbnail video-file)
(get-file-info file-id)

;; 缓存管理
(clear-file-cache &key older-than max-size)
(get-file-cache-stats)
```

**预计工作量**: 4-5 天  
**代码量**: ~500 行

---

#### 1.2 支付和 Stars 系统完善 (高优先级)

**现状**: 已有基础支付模块，需要完善 Telegram Stars 集成

**新增函数**:
```lisp
;; Stars 支付
(create-star-invoice amount &key description payload)
(send-star-payment user-id amount &key message)
(get-star-balance)
(get-star-transactions &key limit offset)

;; 退款
(refund-star-payment payment-id &key reason)

;; 付费媒体
(send-paid-media chat-id media-list star-amount &key caption)
(get-paid-media-post post-id)

;; Giveaway
(create-star-giveaway star-amount winner-count &key duration)
(select-giveaway-winners giveaway-id)
```

**预计工作量**: 5-6 天  
**代码量**: ~600 行

---

#### 1.3 账号和安全增强 (高优先级)

**现状**: 已有基础认证，缺少高级安全功能

**新增函数**:
```lisp
;; QR 码登录
(generate-qr-login-token)
(import-qr-login-token token)
(check-qr-login-status token-id)

;; 隐私设置
(get-privacy-settings)
(set-privacy-setting rule-type allowed-users &key denied-users)
(get-privacy-visibility user-id)

;; 会话管理
(get-active-sessions)
 terminate-session(session-id)
 terminate-all-sessions()

;; 两步验证
(enable-two-factor-auth password &key hint email)
(disable-two-factor-auth password)
(change-two-factor-password old-password new-password)
```

**预计工作量**: 4-5 天  
**代码量**: ~500 行
