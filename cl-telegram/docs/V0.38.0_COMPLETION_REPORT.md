## 使用示例

### Inline Mode 示例

```lisp
;; 创建带切换聊天功能的 inline 按钮
(let ((button (cl-telegram/api:make-inline-keyboard-button-enhanced
               "Select Chat"
               :switch-inline-query-chosen-chat
               (cl-telegram/api:make-switch-inline-query-chosen-chat
                :query "share"
                :allow-groups t
                :allow-channels t))))
  ;; 使用按钮创建 inline 键盘
  )
```

### Stickers 示例

```lisp
;; 建议贴纸集短名
(let ((title "My Awesome Stickers")
      (suggested (cl-telegram/api:suggest-sticker-set-short-name title)))
  ;; suggested => "my_awesome_stickers"
  
  ;; 检查名称是否可用
  (when (cl-telegram/api:check-sticker-set-short-name suggested)
    ;; 创建贴纸集
    ))
```

### Payment 示例

```lisp
;; 完整的支付流程
(let* ((form (cl-telegram/api:get-payment-form "product_123_payload"))
       (form-id (cl-telegram/api:payment-form-id form)))
  ;; 提交支付
  (cl-telegram/api:send-payment-form
   form-id
   "product_123_payload"
   "{\"token\": \"provider_token\"}"
   :name "John Doe"
   :email "john@example.com"))
```
