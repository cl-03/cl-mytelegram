# v0.27.0 开发计划 - 聊天和搜索增强

**版本**: 0.27.0  
**发布日期**: 2026-06-30  
**优先级**: P2 - 增强功能

---

## 一、功能概述

v0.27.0 专注于提升用户体验的聊天管理和搜索发现功能。

### 1.1 自动删除消息

实现 Telegram 风格的定时自毁消息功能。

**核心功能**:
- 设置消息自动删除定时器
- 支持双向自毁消息（双方聊天）
- 支持群组/频道自毁消息
- 可配置的删除时间（1 秒 - 1 周）
- 删除前通知/静默删除选项

### 1.2 聊天备份导出

完整的聊天记录导出/导入功能。

**核心功能**:
- 导出聊天记录为 JSON/HTML
- 包含媒体文件选项
- 增量备份支持
- 导入恢复聊天记录
- 压缩和加密选项

### 1.3 全局搜索

跨聊天的统一搜索功能。

**核心功能**:
- 跨所有聊天搜索消息
- 按发送者过滤
- 按日期范围过滤
- 按媒体类型过滤
- 搜索结果高亮和预览

### 1.4 媒体库/文件库

统一的媒体和文件管理。

**核心功能**:
- 按类型浏览所有媒体（照片/视频/文档/音频）
- 按聊天来源过滤
- 按日期排序
- 搜索文件名/内容
- 批量操作（下载/删除/移动）

### 1.5 个性化主题

深度的 UI 定制功能。

**核心功能**:
- 自定义颜色主题
- 动态主题背景
- 每个聊天独立背景
- 自定义应用图标
- 字体大小调整
