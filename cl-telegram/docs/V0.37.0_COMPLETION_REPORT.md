## 已知问题和限制

### 1. 流式传输 (File Management)
**问题:** `get-file-download-stream` 和 `get-file-upload-stream` 返回 NIL  
**原因:** 需要传输层集成  
**计划:** v0.38.0 或更高版本实现

### 2. QR 码登录 (Account Security)
**问题:** `*qr-login-state*` 变量已定义但完整流程未实现  
**原因:** 需要 MTProto 协议层支持  
**计划:** 等待官方 API 稳定后实现

### 3. 进度回调 (File Management)
**问题:** 文件传输进度已追踪但无 UI 回调机制  
**原因:** 当前实现为轮询模式  
**计划:** v0.38.0 考虑添加回调支持
