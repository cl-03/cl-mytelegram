| 草稿管理 | 保存/获取/清除草稿 | 8 |
| 多媒体消息 | 发送相册/多媒体 | 6 |
| 消息副本 | 复制消息到聊天 | 4 |

**新增函数**: 32+

#### API 示例
```lisp
;; 流式消息
(stream-message chat-id "正在生成..." :stream t)

;; 定时消息
(schedule-message chat-id "Reminder" :schedule-date (+ (get-universal-time) 3600))

;; 保存草稿
(save-draft chat-id "Hello" :reply-to-message-id 123)

;; 发送相册
(send-album chat-id (list photo1 photo2 photo3) :caption "Trip 2025")
```
